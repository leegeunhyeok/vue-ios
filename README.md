<h1 align="center">vue-ios</h1>
<p align="center">üçé iOS Style components library for Vue.js</p>

## Usage

```
npm install vue-ios
```

```html
<template>
  <div id="app">
    <iToggle/>
  </div>
</template>

<script>
import { iToggle } from 'vue-ios'

export default {
  name: 'app',
  components: {
    iToggle
  }
}
</script>
```

## Project setup
```
npm i
```

### Compiles and hot-reloads for development
```
npm run dev
```

### Compiles and minifies for production
```
npm run build
```
